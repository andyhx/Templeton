# Templeton
Render a mustache or glasses over an image of a face

## Build

### clone repo
    git clone https://github.com/kallaballa/Templeton.git
    
### build dlib
    cd third/
    tar -xf dlib-19.0.tar.bz2
    cd dlib-19.0
    sudo python setup.py install
    
### build Templeton
    cd ..
    make
